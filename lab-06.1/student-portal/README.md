Я использую тег Link, чтобы не было необходимости перезагружать страницу при каждом переходе по ссылке. React Router далее сам перерисовывает часть экрана.
Outlet используется как заполнитель, При переходе на /about он превращается в компонент about внутри layout
При переходе на все страницы кроме существующих отрисовывается страница 404, реализовано через path(*)
